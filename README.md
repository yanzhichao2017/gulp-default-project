# gulp自动化初级构建的传统项目
- 可在 /app 文件夹中 自行添加依赖库（例如:jquery,bootstrp,easyui,mui...）
- 文件结构改动请自行配置gulpfile.js

## 初始化项目
```
git clone https://github.com/yanzhichao2017/gulp-default-project.git
```
```
npm install //安装依赖
gulp dev //开发
gulp build //生产
```
## 特性
### gulp dev
- sass
- hot browser
### gulp build
- rar:['js','css','img']
- copy fonts
